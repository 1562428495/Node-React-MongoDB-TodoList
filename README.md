这是一个基于 `Node` + `React` + `Webpack` + `MongoDB` 实现的 ToDoList 应用。

## 功能：

- 添加 ToDo
- 删除 ToDo

## 安装
`npm install`

## 启动
`webpack -w`

`npm start`

`mongod --dbpath D:/mongodb/data`

> dbpath 是本地 MongoDB/data 的路径，关于 MongoDB 的安装和配置，参考博客
